\section*{Scope}
This supplement contains detailed diagnostic tables moved out of the main manuscript to keep the LNCS conference version within the 14-page limit. The k-ablation and forward-split diagnostics use the recovered main fixed-universe return matrix included in the replication package; they are intended as sanity checks rather than byte-for-byte reproductions of the saved Step 4 ranking artifact.
